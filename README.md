# 灵犀 · AI 读心仪 🔮

一束摄像头，读出你的**心率 · 情绪 · 意图** —— 多模态生命体征融合，全程本地运行、画面绝不上传。

> 打开网页 → 允许摄像头 → 对准镜头约 8 秒锁定心率 → 灵犀开始读你的心。
> 试试对它**比个心 🫰、挥挥手 👋、点个赞 👍**，灵犀会回应你。

## ✨ 它能做什么

- **❤️ 摄像头测心率（rPPG）** —— 从面部皮肤的微小血流颜色变化中提取真实脉搏。
  采用 **POS 算法 + FFT 频谱解算**（合成信号测试误差仅 ~0.3 BPM），屏幕上的 ECG 脉搏波和心跳光环都是你真实的心跳。
- **🫁 呼吸 / 🩸 血氧 / 🧘 压力(HRV)** —— 摄像头估算的生命体征仪表盘。
- **😌 情绪融合** —— 表情(MediaPipe FaceLandmarker 478 点 + 52 肌肉信号) × 生理信号，输出「效价 × 唤醒」情绪坐标。
  关键能力：你**面无表情**时，也能靠心率、呼吸、压力读出你嘴上不说的隐藏状态。
- **🔮 意图读心** —— 专注 / 若有所思 / 紧张 / 放松 / 想交流 …… 每次都给出「读心依据」。
- **🖐️ 手势互动** —— 12 种手势（👍✌️✊☝️🖐️🤟👎 + 🫰比心 / 👌OK / 👋挥手 / 🫶双手比心 / 🤔托腮），
  每个手势触发灵犀的暖心回应、特效与能量，附手势图鉴收集。
- **💗 情绪价值** —— 灵犀伴随球暖心话、心跳光环、能量条、成就墙、4-7-8 呼吸教练、读心卡截图导出。

## 🚀 在线使用

部署到 GitHub Pages 后即可用 **HTTPS** 打开（手机也能用摄像头）：

> `https://<用户名>.github.io/<仓库名>/`

## 🖥️ 本地运行

摄像头需要安全上下文，用 `localhost` 最稳：

```bash
cd lingxi
python -m http.server 8646
# 浏览器打开 http://localhost:8646
```

## 🔒 隐私

所有视频画面与计算**只在你的浏览器本地完成，不上传任何服务器**。模型从 CDN 加载，仅此而已。

## ⚠️ 免责声明

本应用为**娱乐 / 自我觉察**用途，所有生理数值（心率、血氧、呼吸、压力）均为摄像头估算，
**非医疗设备，不能用于任何医疗诊断或健康决策**。血氧由普通 RGB 摄像头估算，仅供参考。

## 🛠️ 技术

纯单文件 `index.html`，无构建、无后端。MediaPipe Tasks-Vision（FaceLandmarker + GestureRecognizer）+ 自研 rPPG（POS/FFT）+ 多模态融合。
